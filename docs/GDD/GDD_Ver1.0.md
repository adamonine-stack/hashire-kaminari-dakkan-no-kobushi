# 1.2 ゲームコンセプト

「誰でも遊べるシンプル操作の格闘ゲーム」

本作品は、ストリートファイター初代のようなシンプルな操作性と、
家庭用ゲームらしい達成感を目指した2D格闘アクションゲームである。

プレイヤーが難しいコマンドを覚える必要はなく、
少ないボタンだけで爽快なバトルが楽しめる。

ゲームクリアまで約20分で遊べる内容とし、
何度でも繰り返し挑戦したくなるゲームデザインを採用する。

# 1.3 開発目的

本作品は

「完成させること」

を最優先とする。

新しい機能を増やすことよりも、
Ver.1.0を完成させることを優先する。

Ver.1.0完成後に
Ver.1.1以降で追加要素を検討する。

---

# 1.4 ゲームの特徴

本作品には以下の特徴がある。

・主人公3人

・敵8人＋ラスボス1人

・全9ステージ

・ストーリー性のある格闘ゲーム

・シンプルな操作

・必殺技あり

・投げ技あり

・制限時間なし

・体力が0になった方が敗北

・主人公全員が倒されるとゲームオーバー

・ゲームオーバー時はステージ1から再開

# 1.7 難易度

ゲーム開始時に選択する。

■かんたん

敵HPを減少させる。

敵AIは変更しない。

■ふつう

標準バランス。

■むずかしい

敵HPは標準。

敵AIが積極的に攻撃・ガード・回避を行う。

# 1.10 勝敗条件

各ステージは1ラウンド制。

相手のHPを0にすると勝利。

主人公のHPが0になると敗北。

主人公3人全員が敗北した時点でゲームオーバーとなる。

ゲームオーバー時は第1ステージから再開する。

制限時間は設けない。

引き分けは存在しない。
